GitHub Actions를 제작 후 작업코드 변경!

2026년 7월 5일 일요일

안녕하세요. 오늘 전 GitHub Actions를 사용하기 위해서 몇개의 파일을 만들었습니다. 그 이후 글 작성시 코드 몇개만 올리고 끝나게 되었습니다. 아이패드를 사용하거나 아이폰으로 작업하시는 분들은 꼭 GitHub Actions을 설정하는걸 추천드립니다.

git fetch origin
git pull --rebase origin main

GitHub 최신버전을 다운!

글을 작성 후

git add .
git commit -m "새 글"
git push origin main

GitHub에 업로드

예전(수동 배포)

git pull ↓ 글 작성 ↓ hugo ↓ public 저장소 Push ↓ my-blog 저장소 Push

지금(자동 배포)

git pull ↓ 글 작성 ↓ 저장소 Push ↓ GitHub Actions가 나머지 자동 처리

이제 저는 이 루틴만 지키면 집, 강의실, 아이패드를 번갈아 사용해도 대부분의 충돌을 예방할 수 있습니다.

여러분들도 간단하고 편하게 블로그 만드세요.
