GitHub 코드

2026년 5월 30일 토요일

안녕하세요. GitHub 코드를 항상 잊어버려 코드를 작성합니다.

GitHub에 올리는 폴더에서 Bash 또는 프롬프트, 터미널을 열어서 밑에 내용을 입력.

git pull --rebase origin main

GitHub에 최신 폴더를 내려받습니다.

hugo

전 Hugo 템플릿을 통해서 블로그를 관리하고 있습니다. 그래서 Hugo를 쳐서 입력을 합니다.

cd public
git add .
git commit -m "Update blog"
git push origin main

모든 파일 커밋 준비 상태로 올리는 코드입니다. cd public 블로그가 빌드된/관리되는 폴더로 이동 git add . 그 폴더 안에서 바뀐 파일들을 커밋 준비 git commit -m “Update blog” 변경 내용을 하나의 버전으로 기록 git push origin main 업로드 시작!

cd ..
git add .
git commit -m "Update source"
git push origin main

저는 윗 코드를 한번 더 입력합니다. 이유는 Private, Public 두개를 연결해서 사용하기 때문입니다. 일반적으로 Public만 사용하면 한번만 사용하면 끝입니다.
