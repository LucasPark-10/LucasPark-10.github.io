Python 자료형(Data Type) 이해하기

2026년 7월 2일 목요일

데이터 분석을 시작하기 전에

데이터 분석을 공부하다 보면 가장 먼저 접하게 되는 것이 Python 자료형(Data Type) 입니다.

자료형은 데이터를 어떤 형태로 저장하고 처리할 것인지를 결정합니다.

예를 들어

나이는 숫자로 저장해야 하고,

이름은 문자열,

시험 점수 여러 개는 리스트,

학생 정보는 딕셔너리로 저장하는 것이 일반적입니다.

데이터 분석에서는 적절한 자료형을 사용하는 것이 매우 중요합니다.

Python의 주요 자료형

아래는 Python에서 자주 사용하는 자료형입니다.

1. 숫자형(Number)

숫자형은 계산이 가능한 데이터를 저장합니다.

int (정수)

a = 10
b = -5
print(type(a))

결과

<class 'int'>

float (실수) 소수점을 포함하는 숫자입니다.

a = 3.141592
print(type(a))

결과

<class 'float'>

2. 논리형(Boolean)

참(True) 또는 거짓(False)을 나타냅니다.

a = 20
print(a >= 19)

b = 17
print(b >= 19)

결과

True
False

3. 문자열(String)

문자를 저장하는 자료형입니다.

name = "Lucas"
city = "Seoul"

문자열끼리는 연결할 수도 있습니다.

print(name + " Park")

결과

Lucas Park

4. 리스트(List)

데이터를 여러개 저장할 수 있는 자료형입니다.

데이터 분석에서 가장 많이 사용하는 자료형 중 하나입니다.

예를 들어 여러 학생의 점수나 판매량 데이터를 저장할 때 사용합니다.

scores = [90, 85, 100]
print(scores)

결과

[90, 85, 100]

리스트는 수정이 가능합니다.

scores[0] = 95
print(scores)

결과

[95, 85, 100]

5. 튜플(Tuple)

튜플도 여러 데이터를 저장하지만 수정이 불가능합니다.

값이 변하면 안 되는 데이터에 주로 사용합니다.

point = (10, 20)

6. 딕셔너리(Dictionary)

Key와 Value 형태로 데이터를 저장합니다.

데이터를 키를 이용하여 쉽게 찾을 수 있습니다.

student = {
    "이름": "Lucas",
    "국어": 95,
    "수학": 88
}

student["국어"]

결과

95

7. 집합(Set)

중복을 허용하지 않는 자료형입니다.

데이터 전처리 과정에서 중복 제거를 할 때 자주 활용됩니다.

numbers = {1,2,2,3,3,3}
print(numbers)

결과

결과
{1,2,3}

8. Numpy기초 (행렬 생성) 

import numpy as np

a = np.array([1, 2, 3])
b = np.array([[1,2],[3,4]])

print(a)
print(b)

결과

[1 2 3]

[[1 2]
 [3 4]]

데이터 분석에서는 어떤 자료형을 많이 사용할까요?

실제로 데이터 분석에서는 다음 자료형을 가장 많이 사용합니다.

자료형 사용 빈도
List ★★★★★
Dictionary ★★★★★
String ★★★★★
int / float ★★★★★
Tuple ★★☆☆☆
Set ★★★☆☆
Complex ★☆☆☆☆

마무리

Python 자료형은 데이터 분석의 가장 기본이 되는 개념입니다.

특히 리스트(List) 와 딕셔너리(Dictionary) 는 앞으로 Pandas와 NumPy를 배우면서도 계속 등장하는 중요한 자료형입니다.
