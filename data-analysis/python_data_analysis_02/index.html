Python 조건문·반복문·함수

2026년 7월 3일 금요일

들어가며

오늘은

  • 조건문 (if / elif / else)
  • 반복문 (for / while)
  • 제어문 (break)
  • 함수 (function)

을 정리하였습니다.

데이터 분석의 기초 문법이자, 이후 Pandas와 Numpy를 다룰 때 계속 사용하게 되는 개념들입니다.

1. 조건문 (Conditional Statement)

조건문은 조건에 따라 실행되는 제어문입니다.

조건이 참(True)이냐 거짓(False)이냐에 따라 실행할 코드를 다르게 지정할 수 있습니다.

if / else

height = 180
if height > 175:
    print('175cm 이상입니다. 탑승하세요.')
else:
    print('고객님은 탑승할 수 없습니다.')

결과

175cm 이상입니다. 탑승하세요.

height = 160 이라면 조건이 거짓이므로 else 블록이 실행됩니다.

고객님은 탑승할 수 없습니다.

if / elif / else

조건이 여러 개일 때는 elif를 사용합니다.

위에서부터 순서대로 조건을 검사하며, 하나라도 참이면 그 이후 조건은 검사하지 않습니다.

height = 170
if height > 175:
    print('175cm 이상입니다. 탑승하세요.')
elif height > 165:
    print('부모님과 탑승하세요.')
else:
    print('고객님은 탑승할 수 없습니다.')

결과

부모님과 탑승하세요.

조건문 중첩(nested if)

조건문 안에 또 다른 조건문을 넣어 세분화된 조건을 처리할 수 있습니다.

height = 170
with_parent = 'yes'

if height > 175:
    print('175cm 이상입니다. 탑승하세요.')
elif height > 165:
    if with_parent == 'yes':
        print('부모님과 탑승하세요.')
    else:
        print('부모님과 함께 탑승해야 합니다.')
else:
    print('고객님은 탑승할 수 없습니다.')

165 < height < 175이면서

with_parent = 'yes'일 때 결과

부모님과 탑승하세요.

with_parent = 'no'일 때 결과

부모님과 함께 탑승해야 합니다.

2. 반복문 (Loop)

반복문은 프로그램이 특정 코드를 반복하여 실행하도록 하는 제어문입니다.

Python에는 for문과 while문이 있습니다.

for문과 range()

range(start, end, stride) 형태로 사용하며, range(5)처럼 하나만 넣으면 0부터 4까지(end 미포함) 생성됩니다.

for i in range(5):
    print(i)

결과

0
1
2
3
4

for i in range(5):
    print("안녕하세요")

결과

안녕하세요
안녕하세요
안녕하세요
안녕하세요
안녕하세요

리스트/딕셔너리 순회

리스트 순회

a = [1, 2, 3, 4, 5]
for i in a:
    print(i)

결과

1
2
3
4
5

딕셔너리 순회 (items() 사용)

dic = {'수학': 90, '영어': 100}
for i, j in dic.items():
    print('과목:', i, '점수:', j)

결과

과목: 수학 점수: 90
과목: 영어 점수: 100

enumerate() 로 인덱스와 값 동시에 접근하기

l = ["a", "b", "c"]
for index, value in enumerate(l):
    print(index, value)

결과

0 a
1 b
2 c

Tip. print의 end= 옵션

Python의 print()문은 기본적으로 출력 끝에 줄바꿈(\n)이 포함되어 있습니다.

줄바꿈을 원하지 않는다면 end=''처럼 원하는 문자를 지정할 수 있습니다.

for i in range(4):
    for j in range(2):
        print('*', end='')
    print(i)

결과

**0
**1
**2
**3

while문

while 조건이 참인 동안 반복해서 실행됩니다.

조건이 거짓이 되면 반복문을 빠져나옵니다.

a = 3
while a <= 5:
    a += 1
    print(a)

결과

4
5
6

동작 순서

  1. a = 3 → 조건(a <= 5) 참 → 진입 → a = 4 → 출력
  2. a = 4 → 조건 참 → 진입 → a = 5 → 출력
  3. a = 5 → 조건 참 → 진입 → a = 6 → 출력
  4. a = 6 → 조건(a <= 5) 거짓 → 반복문 종료

3. 제어문 (break)

반복문 중간에 특정 조건이 만족되면 반복을 강제로 멈추고 싶을 때 break를 사용합니다.

for문 - break

a = ["가", "나", "다", "라", "메롱", "마", "바", "사"]
for i in a:
    if i == "메롱":
        print("메롱?")
        break
    print(i)

결과

가
나
다
라
메롱?

while문 - break

while True:처럼 조건이 항상 참이면 무한 반복이 되므로, 반드시 if ~ break문으로 반복을 멈춰줘야 합니다.

a = 3
while True:
    a -= 1
    if a == 0:
        break
    print(a)

결과

2
1

동작 순서

  1. a = 3 → a -= 1 → a = 2 → a == 0 아니므로 통과 → 출력
  2. a = 2 → a -= 1 → a = 1 → a == 0 아니므로 통과 → 출력
  3. a = 1 → a -= 1 → a = 0 → a == 0 참 → break 실행, 반복문 종료

4. 함수 (Function)

함수는 특정 작업을 수행하기 위해 독립적으로 설계된 코드의 집합입니다.

같은 코드를 반복해서 작성하지 않고, 필요할 때마다 호출해서 재사용할 수 있습니다.

def data_science():
    print("Hello World")

data_science()

결과

Hello World

리턴 (return)

return은 함수를 종료하며 결과 데이터를 반환하는 용도로 사용합니다.

return이 없으면 함수는 None을 반환합니다.

def no_return():
    a = 'hi'

re = no_return()
print(re)

결과

None

def do_return():
    return 'hi'

re = do_return()
print(re)

결과

hi

파라미터 (parameter)

파라미터는 함수 호출 시 전달하는 데이터를 받아서 함수 내에서 사용되는 변수입니다.

def data_science(sentence):
    print(sentence)

data_science("매개변수로 전달된 문자열 출력")

결과

매개변수로 전달된 문자열 출력

디폴트 파라미터 (Default Parameter)

함수 호출 시 파라미터에 대한 입력 값이 없으면, 디폴트로 설정된 값이 파라미터 변수로 입력됩니다.

def minus(num1=3, num2=2):
    print(num1 - num2)

minus(2)   # num1=2 (입력값), num2=2 (디폴트값)

여기서 num1=3, num2=2는 **“만약 호출할 때 값을 안 주면 이걸 쓸게”**라는 기본값(디폴트) 설정입니다.

minus(2) 부르면 파라미터가 하나만 전달됐죠. 파이썬은 순서대로 채웁니다.

num1 ← 여기에 2가 들어감 (내가 넣어준 값)

num2 ← 아무 값도 안 줬으니 디폴트값 2가 그대로 들어감

즉 실제로는 이렇게 실행은

pythondef minus(num1=2, num2=2):  # num1은 내가 준 값으로 교체됨
    print(num1 - num2)

그래서 계산은 2 - 2 = 0 → 결과 0이 나옵니다.

결과

0

전역 변수와 지역 변수

  • 전역(global) 변수: 코드 전체에서 사용 가능. 특정 블록 밖에서 선언되어 어디서든 사용 가능한 변수
  • 지역(local) 변수: 특정 블록에서만 사용 가능. 함수 등 블록 내에서 선언되어 그 블록 내에서만 사용 가능

지역 변수를 함수 밖에서 사용하면 오류가 발생합니다.

def print_lv():
    lv = 100        # <-- 지역 변수 (함수 안에서만 존재)
    print(lv)       # <-- 지역 변수 사용 (함수 안이라 OK)

print_lv()
print(lv)   # <-- 오류! lv는 함수 밖에서는 존재하지 않음 (지역 변수라서 함수가 끝나면 사라짐)

결과

100
NameError: name 'lv' is not defined

전역 변수와 같은 이름의 지역 변수를 함수 안에서 선언하면, 함수 안에서는 지역 변수가 우선 사용됩니다.

gv = 10                 # <-- 전역 변수 (파일 전체 어디서든 접근 가능)

def print_gv():
    gv = 100            # <-- 지역 변수 (전역 변수 gv와 이름은 같지만 별개의 변수!)
    print(gv)           # <-- 여기선 지역 변수 gv(=100)가 우선 사용됨

print_gv()
print(gv)               # <-- 함수 밖이므로 전역 변수 gv(=10)가 출력됨 (함수 안의 gv=100은 영향 없음)

결과

100
10
