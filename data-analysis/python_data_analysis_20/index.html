데이터 처리 - 결측치와 이상치

2026년 7월 23일 목요일

들어가며

머신러닝 모델을 만들기 전, 데이터에 존재하는 결측치(Missing Value) 와 이상치(Outlier)를 적절히 처리하는 것은 분석 품질을 좌우하는 핵심 단계입니다.

이번 글에서는 결측치와 이상치의 개념부터 처리 방법까지 정리하였습니다.

사용 도구

구분 도구
데이터 Titanic 데이터셋 (PassengerId, Survived, Pclass 등)
데이터 조작 도구 Pandas
시각화 도구 Matplotlib, Seaborn

Pandas 기초 문법으로는 데이터 선택, 정렬, 카운트, 조작, 집계 등이 있습니다.

  • 데이터 선택: 행 선택, 열 선택, 조건으로 선택
  • 데이터 정렬: 인덱스로 정렬, 값으로 정렬, 오름차순 정렬
  • 데이터 카운트: 카운트, 중복 제거
  • 데이터 조작: 조건을 만족하는 데이터 변경, 커스텀 함수로 조작
  • 데이터 집계: 그룹별 집계, 집계 함수

1. 결측치와 이상치

결측치란?

  • 일부 누락된 데이터
  • 데이터 분석에 방해가 되므로 적절한 처리 필요
  • 분석가의 견해가 반영되어, 분석하는 사람에 따라 다른 결과가 도출될 수 있음

결측치 확인하기

titanic.isnull().sum()

PassengerId    0
Survived       0
Pclass         0
Name           0
Sex            0
Age          177
SibSp          0
Parch          0
Ticket         0
Fare           0
Embarked       2
dtype: int64

이상치란?

  • 다른 데이터와 동떨어진 분포를 가지는 데이터
  • 분석 결과를 편향·왜곡시킴
  • 모델 성능에 악영향을 줄 수 있음

이상치 확인하기 (Boxplot 활용)

%matplotlib inline
boxplot = titanic.boxplot(column=['Age'])

2. 결측치 처리하기

결측치 처리 방법은 크게 수집 → 대체 → 제거 순서로 고려합니다.

방법 설명 특징
수집 결측된 데이터를 추가로 수집 가장 좋은 방법이지만 추가 수집이 불가능한 경우가 많고 비용·시간이 소요됨
대체 적절한 방법으로 다른 값으로 대체 데이터가 적을 때 사용, 데이터에 편향이 생길 수 있음
제거 결측치가 포함된 행 또는 열 삭제 수집·대체가 불가능하거나 데이터가 충분할 때 사용

결측치 대체

  • 행의 정보로 추론하여 대체: 변수 간의 결합으로 다른 변수를 설명할 수 있을 때 사용
  • 열의 정보로 추론하여 대체: 주로 시계열 데이터에서 사용 (예: 서울의 월별 평균/최고/최저 기온 데이터)

여러 가지 결측치 대체 방법

  • 보간법: 근접한 데이터로 유추
  • KNN 알고리즘: 데이터의 유사도로 유추
  • SVM 알고리즘: SVM 머신러닝 알고리즘으로 유추
  • MICE: 하나의 데이터에 결측치가 여러 개인 경우 사용

결측치 제거

  • 행 전체를 삭제: 데이터가 충분히 많은 경우 사용
  • 열 전체를 삭제: 분석에 큰 의미가 없거나 데이터 대다수가 결측된 경우 사용

실습: Titanic 데이터를 활용한 결측치 처리

3. 이상치 처리하기

이상치 처리 방법도 결측치와 유사하게 대체 → 변수화 → 제거 로 접근합니다.

방법 설명 특징
대체 통계적 방법으로 이상치를 다른 값으로 대체 데이터 수가 적을 때 주로 사용
변수화 데이터를 설명할 수 있는 변수로 변환 오류에 의한 이상치가 아닌 경우에 사용
제거 오류에 의한 이상치인 경우 데이터를 삭제 데이터 수가 충분할 때 주로 사용

이상치 대체

  • 행의 정보로 추론하여 대체: 변수 간의 결합으로 다른 변수를 설명할 수 있을 때 사용
  • 열의 정보로 추론하여 대체: 주로 통계값(평균 등)을 이용

이상치 변수화

극단적인 이상치를 구간(bin)으로 변환하여 처리합니다.

예: 문화지리학과 졸업생 연봉 데이터

이름 연봉($) → 변수화된 연봉
… 23,000 → 20k~20k
… 19,000 → 15k~20k
Michael Jordan 630,000 → over 30k
… 30,000 → over 30k

Michael Jordan처럼 극단적으로 튀는 값을 구간화하여 다른 값들과 같은 스케일로 표현할 수 있습니다.

실습: Titanic 데이터를 활용한 이상치 처리

마무리 정리

결측치 처리 방법: 수집 → 대체 → 제거

이상치 처리 방법: 대체 → 변수화 → 제거

데이터 전처리는 머신러닝 모델의 성능을 결정짓는 매우 중요한 단계입니다.

데이터의 특성과 양에 따라 적절한 결측치/이상치 처리 방법을 선택하는 것이 핵심입니다.
