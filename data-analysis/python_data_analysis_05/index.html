Pandas 기초

2026년 7월 6일 월요일

들어가며

오늘은 Pandas개념과 기초를 정리하려고 합니다.

1. Pandas

pandas란?

  • 데이터 조작 및 분석을 위한 Python 라이브러리
  • 구조화된 데이터를 효과적으로 처리·저장 가능
  • Numpy를 기반으로 설계됨
import pandas as pd

Series 생성

  • Series: 1차원 데이터를 저장하는 pandas의 기본 자료구조
  • 하나의 Series 객체 값은 모두 동일한 데이터 타입을 가짐
import pandas as pd

s = pd.Series([1, 3, 5, np.nan, 6, 8])
print(s)
# 0    1.0
# 1    3.0
# 2    5.0
# 3    NaN
# 4    6.0
# 5    8.0
# dtype: float64

0~9까지 랜덤한 5개의 데이터로 Series 생성 

import pandas as pd
import numpy as np

data = pd.Series(np.random.randint(10, size=5))
print(data)

2. Pandas 문법 (1) - Series

Series index 설정

0~9까지 랜덤한 5개의 데이터로 “A”,“B”,“C”,“D”,“E” 안에 넣게 됨 

import pandas as pd
import numpy as np

s = pd.Series(np.random.randint(10, size=5), index=["A","B","C","D","E"])
print(s)

Series 내 index, value 값 확인 

import pandas as pd
import numpy as np

s = pd.Series(np.random.randint(10, size=5), index=["A","B","C","D","E"])
print(s.index)   # Index(['A', 'B', 'C', 'D', 'E'], dtype='object')
print(s.values)  # [2 8 3 0 2]
print(s.A)       # 인덱스 이름으로 바로 접근 가능 → 2

indexing, slicing을 통한 값 확인 

import pandas as pd
import numpy as np

s = pd.Series(np.random.randint(10, size=5), index=["A","B","C","D","E"])
print(s['C'])            # 단일 값 접근
print(s[['B','C','E']])  # 여러 개 선택 (fancy indexing)
print(s[1::2])           # 슬라이싱 (1번부터 2칸씩)

Series의 Broadcasting

  • 모든 원소에 동일 연산이 자동으로 적용됨 (numpy와 동일한 개념)
import pandas as pd
import numpy as np

s = pd.Series(np.random.randint(10, size=5), index=["A","B","C","D","E"])
print(s * 10)  # 모든 값에 10을 곱함

조건에 맞는 데이터 추출 (Boolean indexing) 

import pandas as pd
import numpy as np

s = pd.Series(np.random.randint(10, size=5), index=["A","B","C","D","E"])
print(s[s > 5])  # 5보다 큰 값만 추출
  • 예시 : 반 학생들 점수 Series에서 s[s >= 90]처럼 조건을 걸면 “90점 이상 받은 학생"만 바로 뽑아낼 수 있습니다.

Series와 dictionary

  • dictionary의 key는 Series의 index로, value는 Series의 값으로 변환됨
import pandas as pd

dic = {'D':3, 'F':7, 'E':5}
s = pd.Series(dic)
print(s)

for idx, val in s.items():
    print(idx, val) # D 3, F 7, E 5

Series의 연산

  • 연산 시 동일한 index끼리 연산됨
  • 짝이 되는 index가 없으면 결과는 NaN
import pandas as pd

s1 = pd.Series([1, 2, 3, 4, 5], index=["A","B","C","D","E"])
s2 = pd.Series([10, 20, 30], index=["A","C","E"])

result = s1 + s2
print(result)
# A    11.0
# B     NaN   (s2에 B가 없어서 NaN)
# C    23.0
# D     NaN
# E    35.0

Series 결측치 제거 

print(result.notnull())          # 결측치가 아닌 곳은 True
print(result[result.notnull()])  # 결측치를 제외하고 값만 추출

3. Pandas 문법 (2) - DataFrame

DataFrame 생성

  • DataFrame: 여러 개의 Series가 모여 이루어진 2차원 데이터 타입 (표 형태)
import pandas as pd

df = pd.DataFrame(columns=["Email", "Name"])
df['Name'] = ['doori', 'minsu']
df['Email'] = ['dr@gmail.com', 'ms@gmail.com']
df['Age'] = [20, 21]

print(df)
print(type(df))          # <class 'pandas.core.frame.DataFrame'>
print(type(df['Name']))  # <class 'pandas.core.series.Series'> (컬럼 하나는 Series)

dictionary로부터 DataFrame 생성 

name = ["doori", "minsu"]
email = ["dr@gmail.com", "ms@gmail.com"]
dic = {"Name": name, "Email": email}
df = pd.DataFrame(dic)
print(df)
  • 예시 : 엑셀에서 데이터를 옮겨올 때, 컬럼명을 key로 하고 각 컬럼의 값을 리스트로 넣은 딕셔너리를 만들면 바로 표 형태(DataFrame)로 변환할 수 있습니다.

인덱스 설정 

df = pd.DataFrame(columns=["Email", "Name"], index=["one", "two"])
df["Name"] = ["doori", "minsu"]
df["Email"] = ["dr@gmail.com", "ms@gmail.com"]
df["Age"] = [20, 21]
print(df)

DataFrame의 요소 가져오기 

print(df.index)    # 행 인덱스 목록
print(df.columns)  # 열 이름 목록
print(df.values)   # 전체 값 (2차원 배열)

조건에 맞는 데이터 추출 

print(df[df['Name'] == 'doori'])  # 이름이 'doori'인 행만 추출
  • 예시 : 회원 명단 DataFrame에서 df[df['Age'] >= 20]처럼 조건을 걸면 “성인 회원만” 손쉽게 필터링할 수 있습니다.

iloc — 위치(정수) 기반 인덱싱

  • DataFrame.iloc[row_index, column_index] (숫자 위치로 접근)
print(df.iloc[0, 1])       # 0행 1열 값 하나
print(df.iloc[[0, 3], 1])  # 0행, 3행의 1열 값
print(df.iloc[0:2, :2])    # 0~1행, 0~1열 슬라이싱

loc — 이름(라벨) 기반 인덱싱

  • DataFrame.loc[index_name, column_name] (인덱스/컬럼 이름으로 접근)
print(df.loc[0, "Name"])
print(df.loc[[0, 3], "Name"])
print(df.loc[0:2, :"Name"])
  • iloc vs loc 차이: iloc은 “몇 번째 행/열인가”(0부터 시작하는 정수 위치), loc은 “이름이 무엇인가”(인덱스/컬럼 라벨)로 접근한다는 점이 핵심 차이입니다.

row(행) 추가 

df.loc[5] = {"Email": "boo@gmail.com", "Name": "boo", "Age": 21}
print(df)

column(열) 추가 — 리스트를 이용 

df["Address"] = ["Seoul", "Busan", "Jeju", "Deagu", "Seoul"]
print(df)

shape

  • DataFrame의 (행 개수, 열 개수)를 tuple 형태로 반환
print(df.shape)  # 예: (5, 3)

concat — DataFrame 합치기 

df1 = pd.DataFrame({"Name": ["doori","minsu","hisu"], "Age": [20,21,26]})
df2 = pd.DataFrame({"Name": ["hana","hyun","boo"], "Age": [21,25,20]})

# 위아래로 이어붙이기 (행 방향), reset_index로 인덱스 재정렬
df3 = pd.concat([df1, df2]).reset_index(drop=True)
print(df3)

# 좌우로 이어붙이기 (열 방향)
df4 = pd.concat([df1, df2], axis=1)
print(df4)
  • 예시 : 1월 판매 데이터와 2월 판매 데이터를 하나로 합칠 때는 axis=0(기본값, 위아래로), 같은 사람의 “설문 응답” 표와 “구매 이력” 표를 옆으로 붙일 때는 axis=1을 사용합니다.

Transpose (전치)

  • 행과 열을 서로 바꿔줌
print(df.T)

4. 데이터 집계 및 요약

예시로 사용할 데이터프레임:

import pandas as pd

names = ["apple", "pear", "banana", "strawberry", "mango", "cherry", "melon"]
quantity = [3, 5, 1, None, 1, 5, 5]
fruit_df = pd.DataFrame({'Name': names, 'Quantity': quantity})
Name Quantity
apple 3
pear 5
banana 1
strawberry NaN
mango 1
cherry 5
melon 5

기초 통계 (sum, mean, var, std) 

print('Quantity sum:', fruit_df.Quantity.sum())    # 20.0
print('Quantity mean:', fruit_df.Quantity.mean())  # 3.33...
print('Quantity var:', fruit_df.Quantity.var())    # 3.87 (분산)
print('Quantity std:', fruit_df.Quantity.std())    # 1.97 (표준편차)

기초 통계 (min, max, argmin, argmax) 

print('Quantity min:', fruit_df.Quantity.min())        # 1.0
print('Quantity max:', fruit_df.Quantity.max())        # 5.0
print('Quantity argmin:', fruit_df.Quantity.argmin())  # 2 (최솟값의 위치)
print('Quantity argmax:', fruit_df.Quantity.argmax())  # 1 (최댓값의 위치)

describe() — 통계치 한눈에 요약 

print(fruit_df.describe())
#        Quantity
# count  6.000000
# mean   3.333333
# std    1.966384
# min    1.000000
# 25%    1.500000
# 50%    4.000000
# 75%    5.000000
# max    5.000000

sort_values() — 정렬 

print(fruit_df.sort_values(by='Quantity'))               # 값 기준 오름차순 정렬
print(fruit_df.sort_values(by='Name', ascending=False))  # 이름 기준 내림차순 정렬

rank() — 순위 매기기 

fruit_df['Rank'] = fruit_df.Quantity.rank()               # 오름차순 순위 (동점은 평균 순위)
fruit_df['Rank'] = fruit_df.Quantity.rank(ascending=False) # 내림차순 순위

counts — 개수 세기 

print(fruit_df.count())                     # 컬럼별 결측치 제외 개수
print(fruit_df['Quantity'].value_counts())  # 값별 등장 횟수
print(fruit_df['Name'].value_counts())      # 각 이름은 1개씩 있음을 확인

원소 체크 (in 연산자) 

if 'apple' in list(fruit_df['Name']):
    print('True')
  • 예시 : 장바구니 리스트에 특정 상품이 담겨있는지 확인하거나, 회원 명단에 특정 이름이 있는지 빠르게 검사할 때 유용합니다.

info() — 데이터프레임 구조 요약 

print(fruit_df.info())
# <class 'pandas.core.frame.DataFrame'>
# RangeIndex: 7 entries, 0 to 6
# Data columns (total 2 columns):
#  #   Column    Non-Null Count  Dtype
# ---  ------    --------------  -----
#  0   Name      7 non-null      object
#  1   Quantity  6 non-null      float64

head() / tail() — 앞/뒤 데이터 미리보기 

print(fruit_df.head())     # 기본 5개 출력
print(fruit_df.head(2))    # 개수 지정 가능

print(fruit_df.tail())     # 기본 뒤에서 5개 출력
print(fruit_df.tail(2))    # 개수 지정 가능

isnull() — 결측치 확인 

print(fruit_df.isnull())        # 결측치 여부를 True/False로 표시
print(fruit_df.isnull().sum())  # 컬럼별 결측치 개수 합계
# Name        0
# Quantity    1

unique() — 컬럼의 유일값 확인 

print(fruit_df['Quantity'].unique())  # [ 3.  5.  1. nan]
print(fruit_df['Name'].unique())      # ['apple' 'pear' 'banana' ...]

csv / excel 파일 읽기·쓰기 

# csv로 저장하고 다시 읽기
fruit_df.to_csv('./fruit.csv', index=False)
df = pd.read_csv('./fruit.csv')

# excel로 저장하고 다시 읽기
fruit_df.to_excel('./fruit.xlsx', sheet_name='Sheet1', index=False)
df = pd.read_excel('./fruit.xlsx', sheet_name='Sheet1')
  • index=False를 지정하지 않으면 DataFrame의 인덱스(0,1,2…)까지 파일의 한 열로 같이 저장되니 주의하세요.
