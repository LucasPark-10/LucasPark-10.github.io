들어가며

데이터 탐색부터 시각화까지 정리하였습니다.

1. 탐색적 데이터 분석

1.1 탐색적 데이터 분석(EDA) 과정이란

  • 1977년, 미국 벨 연구소의 존 튜키(John Tukey) 박사에 의해 제안된 개념 — EDA(Exploratory Data Analysis)
  • 가설 검증이나 모형 적용 전에 데이터가 가진 정보를 사람에게 전달하는 방법
  • 적절한 모형, 시각화 산출물을 사용하여 다양한 방식으로 데이터를 표현하고 상관관계를 분석

1.2 탐색적 데이터 분석 과정이 필요한 이유

식재료와 음식의 관계를 비유해 보겠습니다.

아무리 좋은 식재료(원본 데이터)가 있어도, 이를 제대로 손질하고 조리하지 않으면 좋은 음식(분석 결과·모델 성능)이 나올 수 없습니다.

즉, 데이터를 있는 그대로 모델에 넣기 전에, 데이터가 가진 특성과 패턴, 이상치, 결측치 등을 충분히 탐색하고 이해하는 과정이 필요합니다.

1.3 탐색적 데이터 분석의 이점

EDA는 다음과 같은 4단계의 계층 구조로 이해할 수 있습니다.

(아래로 갈수록 기반, 위로 갈수록 최종 목표)

단계설명
성공프로젝트의 성공
의사결정의사결정권자의 의사결정에 도움
이해와 개선데이터에 대한 이해를 바탕으로 프로젝트의 성과 개선
의사소통프로젝트 수행자들의 의사소통을 원활하게 함

즉, EDA는 단순히 데이터를 들여다보는 행위를 넘어 팀 내 의사소통 → 데이터 이해와 개선 → 올바른 의사결정 → 프로젝트 성공으로 이어지는 기반이 됩니다.

1.4 탐색적 데이터 분석의 과정

EDA는 아래 네 단계를 목적을 달성할 때까지 순차적으로 반복하는 과정입니다.

데이터 확인 → 데이터 정제 → 특성 엔지니어링 → 상관관계 분석
  • 데이터 확인: 데이터의 구조, 타입, 크기 등을 파악
  • 데이터 정제: 결측치·이상치·중복 등을 처리
  • 특성 엔지니어링: 기존 데이터를 가공해 새로운 특성(feature)을 만들거나 변환
  • 상관관계 분석: 변수 간 관계를 분석해 인사이트 도출

1.5 탐색적 데이터 분석을 수행하기 위해 필요한 도구

구분도구
데이터(예: PassengerId, Survived, Pclass 등을 담은 타이타닉 데이터 형태)
데이터 조작 도구Pandas
시각화 도구Matplotlib, Seaborn

2. 데이터 처리를 위한 Pandas

2.1 Pandas란

  • 파이썬 라이브러리
  • 구조화된 데이터를 효과적으로 처리하고 저장
  • NumPy를 기반으로 설계되어 빠른 속도를 갖음

2.2 Pandas의 데이터 구조

Pandas는 차원에 따라 세 가지 자료구조를 제공합니다.

2차원DataFrame
3차원Panel

seoul_month_avgtemp = pd.Series([-2.2, -1.1, 7.7, 14.8],

    name='avgTemp')

2.3 탐색적 데이터 분석과 Pandas

Pandas는 (문서 → 표 → 탐색) 흐름, 즉 원본 데이터 로드 → DataFrame 구조화 → 탐색/집계/시각화로 이어지는 EDA 파이프라인의 핵심 도구입니다.

2.4 Pandas로 데이터 불러오기

  • Pandas는 csv, excel, json, html 등 거의 대부분의 데이터 포맷을 지원합니다.
  • 데이터를 DataFrame으로 만드는 데이터 불러오기는 read 계열 함수를 사용합니다. (예: read_csv(), read_excel())
  • DataFrame을 파일로 저장하는 것은 to 계열 함수를 사용합니다. (예: to_csv(), to_excel())

2.5 DataFrame 구조

DataFrame은 크게 두 가지 축으로 이루어집니다.

  • Index: 행(row)을 식별하는 이름표
  • Column: 열(column)을 식별하는 이름표
df['최저기온'].iloc['2022-04']

위와 같이 열 이름(Column)과 위치 기반 인덱싱(iloc)을 조합해 특정 값에 접근할 수 있습니다.

2.6 Pandas 기초 문법

Pandas의 기초 문법은 아래 다섯 가지 흐름으로 정리할 수 있습니다.

데이터 선택 → 데이터 정렬 → 데이터 카운트 → 데이터 조작 → 데이터 집계
단계세부 기능
데이터 선택행선택 / 열선택 / 조건으로 선택
데이터 정렬인덱스로 정렬 / 값으로 정렬 / 오름차순·내림차순 정렬
데이터 카운트카운트 / 중복 제거
데이터 조작조건을 만족하는 데이터 변경 / 커스텀 함수로 조작
데이터 집계그룹별 집계 / 집계 함수

데이터 선택

# 행 선택
df.loc[0]
df.iloc['2022-01']

# 열 선택
df['평균기온']

# 조건으로 선택
df[df['평균기온'] < -2]['평균기온']

데이터 정렬

# 인덱스로 정렬
df.sort_index()

# 값으로 정렬
df.sort_values(by=['평균기온'])

# 오름차순 / 내림차순
df.sort_values(by=['평균기온'], ascending=True)

데이터 카운트

# 카운트
df["날씨"].value_counts()

# 중복 제거
df["날씨"].unique()

데이터 조작

# 조건을 만족하는 데이터 변경
df.loc[df['평균기온'] < 0, '평균기온'] = '영하'

# 커스텀 함수로 데이터 조작
def drop_outlier(num: int):
    if num > 10:
        return 0
    else:
        return num

df['최고기온'] = df['최고기온'].map(drop_outlier)

데이터 집계

# 그룹별 집계
df.groupby(by="날씨")

# 집계 함수
df.groupby(by="날씨")["최고기온"].mean()
df.groupby(by="날씨")["최고기온"].min()
df.groupby(by="날씨")["최고기온"].max()
df.groupby(by="날씨")["최고기온"].median()

3. 시각화를 위한 Matplotlib

3.1 Matplotlib이란

  • 파이썬 라이브러리
  • 데이터를 그래프나 차트로 시각화할 수 있는 라이브러리

3.2 그래프 그려보기

가장 단순한 형태는 plt.plot()을 직접 호출하는 방식입니다.

x = [1, 2, 3, 4, 5]
y = [1, 2, 3, 4, 5]
plt.plot(x, y)
plt.title("First Plot")
plt.xlabel("x")
plt.ylabel("y")

실무에서는 fig, ax = plt.subplots()로 Figure(도화지)와 Axes(그래프 영역)를 명시적으로 분리해서 다루는 방식을 더 많이 사용합니다.

x = [1, 2, 3, 4, 5]
y = [1, 2, 3, 4, 5]
fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title("First Plot")
ax.set_xlabel("x")
ax.set_ylabel("y")

Matplotlib 그래프의 각 구성 요소는 다음과 같이 이름 붙여집니다.

  • Figure: 그래프 전체를 담는 도화지
  • Axes: 실제 그래프가 그려지는 영역 (하나의 Figure 안에 여러 개의 Axes가 있을 수 있음)
  • Title: 그래프 제목
  • x label / y label: 축 이름
  • Major tick / Minor tick: 축의 큰/작은 눈금
  • Grid: 격자선
  • Legend: 범례
  • Line (line plot), Marker (scatter plot): 실제 데이터를 표현하는 요소

3.3 그래프 저장하기

fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title("First Plot")
ax.set_xlabel("x")
ax.set_ylabel("y")
fig.set_dpi(300)
fig.savefig("first_plot.png")

fig.savefig()로 그래프를 이미지 파일로 저장할 수 있으며, set_dpi()로 해상도를 조정할 수 있습니다.

3.4 그래프 여러 개 그리기

plt.subplots(nrows, ncols)에 행/열 개수를 지정하면 여러 개의 Axes를 한 Figure 안에 배치할 수 있습니다.

x = np.linspace(0, np.pi*4, 100)
fig, axes = plt.subplots(2, 1)
axes[0].plot(x, np.sin(x))
axes[1].plot(x, np.cos(x))

3.5 Line plot 꾸미기

ax.plot()은 linestyle, marker, color 등 다양한 스타일 옵션을 지원합니다.

fig, ax = plt.subplots()
x = np.arange(15)
y = x ** 2
ax.plot(
    x, y,
    linestyle=":",
    marker="*",
    color="#524FA1"
)

Line style

x = np.arange(10)
fig, ax = plt.subplots()
ax.plot(x, x, linestyle="-")      # solid
ax.plot(x, x+2, linestyle="--")   # dashed
ax.plot(x, x+4, linestyle="-.")   # dashdot
ax.plot(x, x+6, linestyle=":")    # dotted

Line color

색상은 문자열 이름("r", "green"), 0~1 사이 그레이스케일('0.8'), 헥스코드("#524FA1") 등 다양한 방식으로 지정할 수 있습니다.

x = np.arange(10)
fig, ax = plt.subplots()
ax.plot(x, x, color="r")
ax.plot(x, x+2, color="green")
ax.plot(x, x+4, color='0.8')
ax.plot(x, x+6, color="#524FA1")

3.6 축 경계 조정하기

set_xlim(), set_ylim()으로 각 축이 보여줄 범위를 직접 지정할 수 있습니다.

x = np.linspace(0, 10, 1000)
fig, ax = plt.subplots()
ax.plot(x, np.sin(x))
ax.set_xlim(-2, 12)
ax.set_ylim(-1.5, 1.5)

3.7 범례 추가하기

label 인자로 각 라인의 이름을 지정하고 ax.legend()로 범례를 표시합니다.

fig, ax = plt.subplots()
ax.plot(x, x, label='y=x')
ax.plot(x, x**2, label='y=x^2')
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.legend(
    loc='upper right',
    shadow=True,
    fancybox=True,
    borderpad=2
)

3.8 다양한 그래프 종류

Plot (마커 커스터마이징)

fig, ax = plt.subplots()
x = np.arange(10)
ax.plot(
    x, x**2, "o",
    markersize=15,
    markerfacecolor='white',
    markeredgecolor="blue"
)

Scatter (산점도)

c(색상), s(크기), alpha(투명도) 등을 배열로 넘겨 점마다 다른 스타일을 줄 수 있습니다.

fig, ax = plt.subplots()
x = np.random.randn(50)
y = np.random.randn(50)
colors = np.random.randint(0, 100, 50)
sizes = 500 * np.pi * np.random.rand(50) ** 2
ax.scatter(
    x, y, c=colors, s=sizes,
    alpha=0.3
)

Bar chart (막대그래프)

# 단순 막대그래프
x = np.arange(10)
fig, ax = plt.subplots(figsize=(12, 4))
ax.bar(x, x*2)

bottom 인자를 활용하면 누적 막대그래프(stacked bar chart)를 그릴 수 있습니다.

x = np.random.rand(3)
y = np.random.rand(3)
z = np.random.rand(3)
data = [x, y, z]

fig, ax = plt.subplots()
x_ax = np.arange(3)
for i in x_ax:
    ax.bar(x_ax, data[i],
        bottom=np.sum(data[:i], axis=0))
ax.set_xticks(x_ax)
ax.set_xticklabels(["A", "B", "C"])

Histogram (히스토그램)

bins로 구간(막대) 개수를 지정해 데이터의 분포를 확인할 수 있습니다.

fig, ax = plt.subplots()
data = np.random.randn(1000)
ax.hist(data, bins=50)

3.9 Seaborn — 통계 시각화 라이브러리

  • Matplotlib 기반의 파이썬 라이브러리
  • Statsmodels의 통계 기능 활용
  • Matplotlib에서 기본 제공하지 않는 통계용 차트 기능을 제공

rugplot

데이터 값의 위치를 축 위에 짧은 선(rug, 카펫)으로 표시해 데이터의 밀집도를 직관적으로 보여줍니다.

x = iris.petal_length.values

sns.rugplot(x)
plt.title("Iris 데이터 중, 꽃잎의 길이에 대한 Rug Plot")
plt.show()

kdeplot

커널 밀도 추정(Kernel Density Estimation)을 이용해 데이터의 분포를 부드러운 곡선으로 표현합니다.

sns.kdeplot(x)
plt.title("Iris 데이터 중, 꽃잎의 길이에 대한 Kernel Density Plot")
plt.show()

distplot

히스토그램(hist) + 커널 밀도 추정(kde) + 러그플롯(rug)을 한 번에 겹쳐서 보여줍니다.

sns.distplot(x, kde=True, rug=True)
plt.title("Iris 데이터 중, 꽃잎의 길이에 대한 Dist Plot")
plt.show()

jointplot

두 변수 간의 산점도와 함께 각 변수의 분포(히스토그램)를 동시에 보여주는 그래프입니다.

sns.jointplot(x="sepal_length", y="sepal_width", data=iris)
plt.suptitle("꽃받침의 길이와 넓이의 Joint Plot", y=1.02)
plt.show()

pairplot

데이터셋에 있는 모든 수치형 변수 쌍에 대해 산점도(또는 히스토그램)를 격자 형태로 한 번에 그려주는, 다변량 탐색에 매우 유용한 함수입니다.

sns.pairplot(iris)
plt.title("Iris Data의 Pair Plot")
plt.show()

마무리 정리

머신러닝 프로젝트의 출발점인 탐색적 데이터 분석(EDA)의 개념과 필요성을 살펴본 뒤, 이를 실제로 수행하기 위한 두 핵심 도구인 Pandas(데이터 조작)와 Matplotlib/Seaborn(시각화)의 기초 문법을 정리했습니다.

핵심 포인트를 정리하면 다음과 같습니다.

  1. EDA는 데이터 확인 → 정제 → 특성 엔지니어링 → 상관관계 분석을 목적 달성까지 반복하는 과정이며, 궁극적으로 팀의 의사소통과 의사결정, 나아가 프로젝트 성공으로 이어집니다
  2. Pandas는 Series(1차원) → DataFrame(2차원) → Panel(3차원) 구조를 중심으로, 데이터 선택·정렬·카운트·조작·집계라는 다섯 가지 기초 문법으로 데이터를 자유자재로 다룰 수 있게 해줍니다.
  3. Matplotlib는 Figure와 Axes를 중심으로 한 세밀한 커스터마이징이 가능하고, Seaborn은 그 위에서 rugplot·kdeplot·distplot·jointplot·pairplot 같은 통계적으로 풍부한 시각화를 손쉽게 제공합니다.
