EDA(탐색적 데이터 분석) 정리 - 시각화부터 전처리까지

2026년 7월 8일 수요일

EDA(탐색적 데이터 분석) 정리

이번 글에서는 EDA(Exploratory Data Analysis) 의 개념부터 특성 엔지니어링까지 실습 코드와 함께 정리하려고 합니다.

1. 데이터 시각화

EDA(Exploratory Data Analysis) 는 수학자 존 튜키가 제안한 개념으로, 데이터를 다양한 방법으로 확인하고 이해하는 과정입니다. 시각화를 통해 자료를 직관적으로 바라볼 수 있으며, 여기서 얻은 지식은 이후 통계 분석이나 모델링에 활용됩니다.

데이터 시각화란 그래프, 차트, 맵과 같은 시각적 요소를 사용해 데이터를 나타내는 프로세스로, 수치 데이터만으로는 발견하지 못한 패턴이나 인사이트를 찾도록 도와줍니다.

주요 시각화 방법

방법 용도 장점 한계
히스토그램 단일 수치형 변수의 분포 계급별 자료 수를 쉽게 파악 원래 값 상실 가능, 연속 데이터 전용
박스 플롯 이상치 식별, 분산 파악 분포 요약, 그룹 비교, 이상치 강조 세부 분포를 보여주지 못함
산점도 두 변수 간 관계/추세 생성·해석 쉬움, 패턴·추세·이상치 표현 범주형 데이터엔 부적합, 포인트 많으면 해석 어려움
막대 차트 범주형 변수의 빈도/비율 이산 카테고리 비교에 효과적 연속 데이터엔 부적합, 카테고리 많으면 혼잡
히트맵 변수 간 상관관계 대규모 데이터에서 패턴·군집 파악 용이 색상 척도에 따라 해석이 왜곡될 수 있음

박스 플롯의 주요 지표:

  • Q1(제1사분위수): 데이터의 25% 지점
  • 중앙값: 데이터 세트를 절반으로 나누는 값
  • Q3(제3사분위수): 데이터의 75% 지점
  • 최댓값(휘스커 상한, 이상치 판단 경계): Q3 + 1.5 × IQR — 이 값보다 큰 데이터는 이상치로 표시됨
  • 최솟값(휘스커 하한, 이상치 판단 경계): Q1 − 1.5 × IQR — 이 값보다 작은 데이터는 이상치로 표시됨

2. Matplotlib

파이썬에서 가장 널리 사용되는 데이터 시각화 라이브러리입니다.

import matplotlib.pyplot as plt

# 점 (1,3), (2,7), (3,9), (4,12) 그리기
# 'ro'는 빨간색('red') 원형 마커('o')를 의미함
plt.plot([1, 2, 3, 4], [3, 7, 9, 12], 'ro')
plt.axis([0, 5, 0, 15])
plt.show()

여러 그래프를 한 번에 나타낼 수도 있습니다.

import numpy as np
import matplotlib.pyplot as plt

time = np.arange(0, 10, 0.5)

plt.plot(time, time, 'ro', time, time**2, 'b--', time, time**3, 'gs')
plt.show()
  • 한계 : Matplotlib은 Seaborn, Plotly 같은 최신 라이브러리에 비해 그래프가 덜 세련되어 보일 수 있습니다.

3. Seaborn

Seaborn은 Matplotlib을 기반으로 하며, 더 세련되고 통계적인 시각화를 간단한 코드로 만들 수 있습니다.

Seaborn의 기능과 장점

  • Matplotlib 기반: 복잡한 시각화를 더 쉽게 생성
  • 세련된 시각화: 미적으로 다듬어진 다양한 테마 포함
  • 복잡한 코드의 단순화: 짧은 구문으로 복잡한 플롯 구현
  • Pandas와의 통합: DataFrame을 직접 다루고 시각화하기 쉬움
  • 고급 통계 플롯: 통계 관계를 보여주는 다양한 고급 플롯 함수 포함

예시 1: 박스 플롯 (Flights 데이터셋) 

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# seaborn에서 자체 제공하는 flights 데이터셋 로드
flights = sns.load_dataset('flights')

plt.figure(figsize=(6, 6))
sns.boxplot(data=flights, x='year', y='passengers')
plt.show()

예시 2: 산점도 (Tips 데이터셋) 

import seaborn as sns
import matplotlib.pyplot as plt

# 레스토랑 손님의 팁 데이터셋
tips = sns.load_dataset("tips")

plt.figure(figsize=(5, 5))
sns.scatterplot(data=tips, x="total_bill", y="tip", hue="time")
plt.show()

4. 데이터 비교

데이터 비교란 두 개 이상의 데이터 집합을 분석·평가하여 유사성, 차이, 추세, 관계 등을 파악하는 과정입니다.

  • 질적 비교: 범주형 데이터의 차이와 유사성을 분석
  • 양적 비교: 수치 데이터의 상관관계 또는 패턴을 분석

예시: 연도별/월별 승객 수 비교 (색상 강조) 

import matplotlib.pyplot as plt
import seaborn as sns

flights = sns.load_dataset('flights')
sns.set(font_scale=1.5)
plt.figure(figsize=(10, 10))

# 특정 연도(1955, 1958)만 강조하여 비교
flights_years = sorted(list(set(flights['year'])))
color = ['lightgrey' if x not in [1955, 1958] else "#4472C4" for x in flights_years]

sns.barplot(data=flights, x="year", y="passengers", palette=color, errorbar=None)
plt.show()

예시: 연도별 승객 수 변화 라인 비교 

import matplotlib.pyplot as plt
import seaborn as sns

flights = sns.load_dataset('flights')

# 연도마다 월별 승객 수 변화 시각화
for year in flights['year'].unique():
    flights_for_year = flights[flights['year'] == year]
    # 1955년과 1958년만 파란색으로 강조
    line_color = '#4472C4' if year in [1955, 1958] else '#EFF3F9'
    plt.plot(flights_for_year['month'], flights_for_year['passengers'], color=line_color)

plt.show()

5. 상관관계 분석

상관관계 분석은 두 양적 변수 간 선형 관계의 강도와 방향을 평가하는 것을 의미합니다.

공분산(Covariance)

두 변수의 공동 변동성을 설명하는 척도로, 두 변수가 함께 증가/감소하면 양수, 반대로 움직이면 음수가 됩니다.

$$Cov(X, Y) = \frac{\sum_{i=1}^{n}(X_i - \bar{X})(Y_i - \bar{Y})}{n}$$

⚠️ numpy.cov()는 기본적으로 표본공분산(n이 아닌 n−1로 나눈 값)을 반환합니다. 아래 예제 결과(114.7)도 n−1 기준입니다.

import pandas as pd
import numpy as np

df = pd.DataFrame({'키': [158, 163, 170, 183, 180], '몸무게': [49, 52, 60, 80, 83]})
print(np.cov(df['키'], df['몸무게']))
  • 공분산의 한계 : 단위에 따라 값이 크게 달라집니다. 예를 들어 키를 cm에서 m로 바꾸면 공분산 값 자체가 크게 변해, 변수 간 관계 강도를 직관적으로 비교하기 어렵습니다.

상관계수(Correlation Coefficient)

단위에 영향받지 않도록 공분산을 표준편차로 나눈 값으로, -1 ~ 1 사이의 값을 가집니다.

$$Corr(X, Y) = \frac{Cov(X, Y)}{\sqrt{\frac{\sum(X-\mu)^2}{N}} \cdot \sqrt{\frac{\sum(Y-\mu)^2}{N}}}$$

  • -1: 완전한 음의 상관관계
  • 0: 독립(관계 없음)
  • 1: 완전한 양의 상관관계
import pandas as pd
import numpy as np

df = pd.DataFrame({'키': [158, 163, 170, 183, 180], '몸무게': [49, 52, 60, 80, 83]})
print(np.corrcoef(df['키'], df['몸무게'])[0][1])
# 단위를 cm -> m로 바꿔도 상관계수는 동일하게 유지됨

6. 상관관계 시각화

상관관계를 시각화하면 두 변수 사이 관계의 강도와 방향을 쉽게 식별할 수 있고, 필요한 특징(feature) 선택에도 도움이 됩니다.

예시로 Scikit-learn의 당뇨병(diabetes) 데이터셋을 사용합니다.

from sklearn import datasets

data = datasets.load_diabetes()
print(data.keys())
print(data['feature_names'])

산점도로 두 변수 관계 보기 

import matplotlib.pyplot as plt

plt.scatter(data['data'][:, 2], data['target'], alpha=0.5)  # BMI vs target
plt.title("Scatter plot")
plt.xlabel('BMI')
plt.ylabel('TARGET')
plt.show()

여러 변수 간 산점도 (pairplot) 

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn import datasets

data = datasets.load_diabetes()
df = pd.DataFrame(data['data'], index=data['target'], columns=data['feature_names'])
df = df.reset_index(drop=True)

plt.figure(figsize=(14, 8))
sns.pairplot(df[['age', 'bmi', 'bp']])
plt.show()

히트맵으로 전체 변수 상관관계 확인 

import seaborn as sns
import matplotlib.pyplot as plt
from sklearn import datasets
import pandas as pd

data = datasets.load_diabetes()
df = pd.DataFrame(data['data'], index=data['target'], columns=data['feature_names'])

sns.heatmap(df.corr())
plt.title('Heatmap by sns.heatmap()', fontsize=20)
plt.show()

Matplotlib만으로도 유사하게 그릴 수 있습니다.

import numpy as np
import matplotlib.pyplot as plt

plt.pcolor(df.corr())
plt.xticks(np.arange(0.5, len(df.columns), 1), df.columns)
plt.yticks(np.arange(0.5, len(df.columns), 1), df.columns)
plt.title('Heatmap by plt.pcolor()', fontsize=20)
plt.colorbar()

7. 데이터 스케일링

데이터 스케일링은 서로 다른 변수의 값 범위를 일치시키는 작업입니다.

필요성

  • 모든 변수가 동일한 범위가 되어 쉽게 비교 가능
  • 규모가 큰 변수의 영향이 과도하게 커지는 것을 방지
  • 이상치 처리에 효과적

스케일링 종류

종류 설명
Min-Max 스케일링 최솟값·최댓값을 이용해 0~1 사이 값으로 변환
표준화 (Standardization) 평균 0, 표준편차 1이 되도록 변환
로버스트 스케일링 중앙값 0, 사분위범위(IQR) 1이 되도록 변환 (이상치에 강함)

예시 데이터로 Scikit-learn의 유방암(breast cancer) 데이터셋을 사용합니다.

표준화 (StandardScaler) 

from sklearn.datasets import load_breast_cancer
from sklearn.preprocessing import StandardScaler

cancer = load_breast_cancer()
X = cancer.data

std = StandardScaler()
std.fit(X)
X_scaled = std.transform(X)

print('Data before scaling')
print(X)
print('Data after scaling')
print(X_scaled)

Min-Max 스케일링 

from sklearn.preprocessing import MinMaxScaler

mms = MinMaxScaler()
mms.fit(X)
X_scaled = mms.transform(X)

로버스트 스케일링 

from sklearn.preprocessing import RobustScaler

rbs = RobustScaler()
rbs.fit(X)
X_scaled = rbs.transform(X)

8. 데이터 전처리

데이터 전처리란 데이터를 분석하기에 효과적인 형식으로 정리·변환·구성하는 작업입니다.

  • 데이터 정제: 누락된 값, 잡음, 일관성 없는 데이터 처리
  • 특징 엔지니어링: 기존 특징에서 새로운 특징을 생성·변환
  • 불균형 데이터 처리: 클래스 불균형이 심하면 모델에 편향이 생기므로 오버샘플링·언더샘플링 등으로 균형을 맞춤

예시로 Seaborn의 titanic 데이터셋을 사용합니다.

import seaborn as sns

titanic = sns.load_dataset('titanic')
print(titanic.info())

불필요한 열 제외 

# 누락된 데이터가 많은 열 제거 (예: deck)
cols = ['deck']
titanic = titanic.drop(cols, axis=1)
print(titanic.info())

선형보간으로 결측치 채우기 

# 누락된 데이터를 선형보간을 이용해 메꿈
titanic['age'] = titanic['age'].interpolate()
print(titanic.info())

9. 특성 엔지니어링

특성 엔지니어링(Feature Engineering) 은 기존 특성에서 새로운 특성을 선택·변환·생성하여 모델의 성능을 향상시키는 과정입니다.

주요 방법

방법 설명
특성 결합 두 개 이상의 기존 특성을 결합해 새로운 특성 생성
범주형 변수 인코딩 문자열 등의 범주형 특성을 수치형 특성으로 변환
일변량 비선형 변환 제곱항, 세제곱항 등을 추가
새로운 특성 생성 도메인 지식을 바탕으로 새로운 유용한 특성을 만듦

예시: 범주형 변수 인코딩 및 특성 결합 (상관관계 계산) 

import numpy as np
import pandas as pd
import seaborn as sns

titanic = sns.load_dataset('titanic')

# 생존 특성을 0과 1로 인코딩하여 상관관계 확인
titanic['alive_1'] = np.where(titanic['alive'] == 'yes', 1, 0)
titanic_sub = titanic[['fare', 'alive_1']]
print(titanic_sub.corr())

예시: 새로운 특징 생성 

import numpy as np
import seaborn as sns

titanic = sns.load_dataset('titanic')

# 요금이 50을 넘으면 비싼 표(1), 아니면 0으로 새로운 칼럼 생성
titanic['expensive'] = np.where(titanic['fare'] > 50, 1, 0)

# 퀸즈타운 승선이면 'Oceania', 아니면 'Europe'으로 대륙 특성 생성
titanic['continent'] = np.where(titanic['embark_town'] == 'Queenstown', 'Oceania', 'Europe')

print(titanic[['expensive', 'continent']])

마무리

EDA는 모델링 이전에 데이터의 특성과 패턴, 이상치, 변수 간 관계를 파악하는 필수적인 과정입니다.

  • 데이터 로드 → 시각화(분포/이상치/관계 파악) → 데이터 비교 → 상관관계 분석 → 스케일링 → 전처리(결측치 등) → 특성 엔지니어링
