데이터 처리 (CSV, Pandas, 병합, 기초 통계)

2026년 7월 7일 화요일

들어가며

오늘은 데이터 이론부터 로드, 변환, 요약, 이상치/결측치 처리, 병합, 기초 통계까지 데이터 처리 전 과정을 정리하려고 합니다.

1. 데이터 이론

데이터란?

  • 정보를 전달하는 연속 혹은 불연속 값의 모음
  • 형식에 따라 정형 데이터, 반정형 데이터, 비정형 데이터로 구분
종류설명예시
정형 데이터표준화되고 명확하게 정의되며, 전통적인 데이터베이스로 처리가 용이한 데이터직원 데이터베이스, 판매 기록
반정형 데이터일정 수준의 표준화와 구조가 있으나 구조에 유연성을 가지는 데이터JSON, XML
비정형 데이터미리 정의된 형식이 없어 전통적인 데이터베이스로 처리가 어려운 데이터사진, 소셜 미디어 게시물
  • 예시 : 엑셀 표나 회원 DB처럼 열/행이 딱 정해진 건 정형, API 응답으로 자주 받는 JSON은 반정형, 유튜브 영상이나 인스타그램 사진은 비정형 데이터에 해당합니다.

2. 데이터 로드

CSV (Comma-Separated Values)

  • 쉼표(,)로 구분한 텍스트 데이터 및 텍스트 파일
  • 저장용량이 작고 단순하며, 다양한 소프트웨어·언어에서 광범위하게 지원되어 데이터 교환에 널리 사용됨

CSV 모듈로 읽기

import csv

# CSV 파일 열기
with open('data.csv', 'r') as file:
    # CSV reader 객체 생성
    csv_reader = csv.reader(file)

    # CSV 파일의 각 행 순회
    for row in csv_reader:
        print(row)  # 각 행의 값들은 리스트로 표현됩니다.

Pandas 라이브러리

  • 데이터 조작 및 분석을 위해 설계된 강력한 파이썬 라이브러리
  • 기존 CSV 모듈에서 수동으로 처리해야 했던 파싱 및 데이터 변환을 자동으로 처리하여, 구조화된 표 형태의 데이터를 작업하는 데 유용함
  • pandas.read_csv() 함수는 CSV 파일을 DataFrame으로 반환
import pandas as pd

# CSV 파일을 DataFrame으로 로드
df = pd.read_csv('data.csv')

# DataFrame 출력
print(df)
  • 예시 : CSV 모듈은 한 행씩 리스트로 읽어와서 숫자 계산 전에 직접 형변환을 해줘야 하지만, Pandas는 read_csv() 한 줄로 숫자·문자 타입까지 자동으로 인식해주기 때문에 실무에서는 대부분 Pandas를 사용합니다.

3. 데이터 변환

기존 데이터 기반 새로운 열 추가

import pandas as pd

df = pd.DataFrame({
    'Name': ['John', 'Sarah', 'Michael'],
    'Age': [31, 26, 29],
    'Occupation': ['Engineer', 'Doctor', 'Teacher']
})

# 'Adult' 열 추가
df['Adult'] = df['Age'] >= 18
print("Adult열 추가!")
print(df)

# 'Birth(years)' 열 추가
df['Birth(years)'] = 2023 - df['Age']
print('Birth열 추가!')
print(df)

범주형 데이터 → 수치형 데이터 변환

import numpy as np

df['Adult'] = np.where(df['Adult'] == True, 1, 0)
print(df)

rename() — 컬럼명 / 인덱스명 변경

import pandas as pd

df = df.rename(columns={'Age': 'Age(나이)'})
print(df)

# 인덱스 이름 변경도 가능
df = df.rename(index={1: 'one'})
print(df)

조건에 따른 행 필터링

import pandas as pd

# 열 조건에 따른 필터링
filtered_df = df[df['Age(나이)'] >= 27]
print(filtered_df)

print('-' * 55)

# 두 가지 조건을 이용한 필터링
filtered_two_df = df[(df['Age(나이)'] >= 27) & (df['Occupation'] == 'Engineer')]
print(filtered_two_df)
  • 예시 : 회원 DB에서 df[(df['Age'] >= 20) & (df['City'] == '서울')]처럼 조건을 여러 개 걸면, “20세 이상이면서 서울에 사는 회원"만 한 번에 뽑아낼 수 있습니다. &는 AND, |는 OR 조건이라는 점을 함께 기억해두면 좋습니다.

4. 데이터 요약하기

  • 요약된 데이터 정보를 통해 데이터의 분포 및 주요 경향을 빠르게 이해할 수 있으며, 이를 통해 데이터간 비교, 정리, 검증 등이 가능해짐

titanic 데이터셋

  • 데이터 분석과 머신러닝 세계에서 가장 잘 알려지고 널리 사용되는 데이터셋 중 하나
  • 1912년 항해 중 침몰한 RMS 타이타닉호의 승객에 관한 정보를 포함
  • 일반적으로 다양한 특징을 기반으로 승객의 생존 여부를 예측하는 모델을 만드는 작업에 사용
import pandas as pd

df = pd.read_csv('./train.csv')
print(df)

head() — 앞 데이터 미리보기

  • 전체 데이터의 첫 5줄에 대한 정보를 제공하며, 5줄 미만인 경우 모든 정보를 제공
df.head()

describe() — 통계치 한눈에 요약

  • DataFrame의 통계 요약을 가져오는 데 사용됨
  • 결측치(NaN)는 제외되고 데이터 요약이 수행됨
print(df.describe())

info() — 데이터프레임 구조 요약

  • 데이터 수, 열 이름, 데이터 유형, 인덱스 범위 및 메모리 사용량과 같이 데이터프레임에 대해 알아야 할 모든 정보를 보여줌
df.info(verbose=True)

shape — 행/열 개수 확인

  • DataFrame의 차원을 나타내는 튜플을 반환
  • 반환 값의 첫 번째 요소는 행의 수, 두 번째 요소는 열의 수
print(df.shape)  # (891, 12)

value_counts() — 값별 등장 횟수

print(df['Survived'].value_counts())
print(df['Pclass'].value_counts())
  • 예시 : 설문조사 데이터에서 df['만족도'].value_counts()를 쓰면, “매우 만족/만족/보통/불만족” 응답이 각각 몇 명씩인지 순식간에 집계할 수 있어서, 막대그래프를 그리기 전 단계로 자주 활용됩니다.

5. 이상치와 결측치

데이터 시각화를 통해 이상치 찾기

import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

sns.boxplot(y=df['Age'])
plt.show()

이상치 처리 — IQR(사분위수 범위) 이용

Q1 = df['Age'].quantile(q=0.25)
Q3 = df['Age'].quantile(q=0.75)

IQR = Q3 - Q1
M = Q3 + 1.5 * IQR  # 이 값을 넘으면 이상치로 판단

print(M)  # 64.8125

# 이상치를 상한값으로 대체
df.loc[df['Age'] > M, 'Age'] = M

sns.boxplot(y=df['Age'])
plt.show()

notnull() — 결측치가 아닌 곳 확인

print(df.notnull())
print(df.notnull().sum())  # 컬럼별 결측치가 아닌 데이터 개수

fillna() — 특정 값으로 결측치 대체

print('Age mean: ', df['Age'].mean())
print('Age max: ', df['Age'].max())
print('-' * 20)

df['Age'] = df['Age'].fillna(df['Age'].mean())
print(df.notnull().sum())

drop() — 특정 열 제거

df = df.drop(['Cabin'], axis=1)
print(df.notnull().sum())

dropna() — 결측치가 있는 행 제거

df = df.dropna()
print(df.notnull().sum())
  • 예시 : Cabin처럼 결측치가 너무 많은 열(891개 중 681개가 결측)은 drop()으로 열 자체를 지우고, Embarked처럼 결측치가 아주 적은 열(2개뿐)은 dropna()로 해당 행만 지우는 식으로, 결측 비율에 따라 다른 전략을 쓰는 것이 일반적입니다.

6. 데이터 병합

concat() — 위아래(행) 또는 좌우(열)로 이어붙이기

import pandas as pd

jul_df = pd.read_csv('./7월.csv')
aug_df = pd.read_csv('./8월.csv')

concat_df = [jul_df, aug_df]

# 위아래로 이어붙이기 (행 방향, 기본값)
jul_aug_df = pd.concat(concat_df).reset_index(drop=True)
print(jul_aug_df)

# 좌우로 이어붙이기 (열 방향)
jul_aug_df_h = pd.concat(concat_df, axis=1).reset_index(drop=True)
print(jul_aug_df_h)

merge() — 고유값(key) 기준 병합

  • 두 DataFrame을 각 데이터에 존재하는 고유값(key)을 기준으로 병합할 때 사용
import pandas as pd

age_sex_df = pd.read_csv('./나이_성별.csv')
add_df = pd.read_csv('./주소.csv')

merge_data = pd.merge(age_sex_df, add_df, on='이름')
print(merge_data)

merge()의 how 옵션

  • how='outer': 양쪽 데이터 모두 유지 (없는 값은 NaN)
  • how='left': 첫 번째 데이터프레임을 기준으로 병합
  • how='right': 두 번째 데이터프레임을 기준으로 병합
# outer: 양쪽 데이터 모두 유지
merge_outer = pd.merge(age_sex_df, add_df, how='outer', on='이름')

# left: age_sex_df 기준
merge_left = pd.merge(age_sex_df, add_df, how='left', on='이름')

# right: add_df 기준
merge_right = pd.merge(age_sex_df, add_df, how='right', on='이름')
  • 예시 : 회원 정보 테이블과 주문 내역 테이블을 merge()로 합칠 때, “모든 회원을 다 보여주되 주문이 없으면 빈 칸으로” 두고 싶으면 how='left'(회원 테이블 기준), “실제 주문이 있는 데이터만” 보고 싶으면 how='inner'(기본값)를 사용하는 식으로 상황에 맞게 고를 수 있습니다.

7. 기초 통계

통계 분석

  • 데이터에서 유용한 정보를 추출하기 위해 사용되는 방법
  • 크게 기술 통계 분석과 추론 통계 분석으로 나눌 수 있음

기술 통계 분석

  • 데이터를 간결하고 요약된 정보로 변환하여 데이터 세트에 대한 이해를 돕기 위해 사용됨
  • 대표적인 방법: 평균값 계산, 분산 계산 등

추론 통계 분석

  • 샘플 데이터(일부 데이터)로부터 전체 데이터를 추론하기 위한 통계
  • 가장 큰 전제는 선택된 표본이 모집단을 대표해야 한다는 것

변수의 종류

  • 질적 변수: 명목형(순서 없음, 예: 성별), 순서형(순서 있음, 예: 학년)
  • 양적 변수: 이산형(셀 수 있음, 예: 자녀 수), 연속형(측정값, 예: 키)

평균값

import pandas as pd

df = pd.read_csv('./train.csv')
print(df.mean())

중앙값

print(df.median())

분산

print(df.var())

표준편차

print(df.std())
  • 예시 : 반 학생들의 시험 점수에서 평균이 70점으로 같아도, 표준편차가 크면 “잘하는 학생과 못하는 학생 차이가 큰 반"이고 표준편차가 작으면 “다들 비슷하게 70점 근처에 몰려있는 반"이라는 의미로 해석할 수 있습니다. 즉 평균만으로는 안 보이는 “데이터가 퍼진 정도"를 표준편차/분산이 보여줍니다.
